在二叉树的遍历方法中，先序遍历、中序遍历和后序遍历是最基本的三种遍历策略，它们都是深度优先搜索（DFS）的应用。这三种遍历方式的区别在于访问根节点的时机。

假设有一个二叉树如下：

```
    A
   / \
  B   C
 / \
D   E
```

### 先序遍历（Pre-order Traversal）
先访问根节点，然后递归地先序遍历左子树，再递归地先序遍历右子树。

遍历顺序为：`A -> B -> D -> E -> C`

### 中序遍历（In-order Traversal）
先递归地中序遍历左子树，然后访问根节点，最后递归地中序遍历右子树。

遍历顺序为：`D -> B -> E -> A -> C`

### 后序遍历（Post-order Traversal）
先递归地后序遍历左子树，然后递归地后序遍历右子树，最后访问根节点。

遍历顺序为：`D -> E -> B -> C -> A`

每种遍历方法都有其特定的应用场景。例如，先序遍历可以用于打印一个结构化的文档，中序遍历在二叉搜索树上可以按顺序访问元素，后序遍历常用于删除或释放节点资源。
